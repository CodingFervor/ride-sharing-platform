# Ride-Sharing Mobility Platform

## English | [中文](#中文)

A ride-sharing and mobility platform built with Go, Gin, PostgreSQL, and Redis.

### Features
- Ride booking and matching
- Real-time GPS tracking
- Driver and vehicle management
- Fare calculation (dynamic pricing)
- Payment and wallet system
- Rating and review system
- Route optimization
- Surge pricing
- Trip history and receipts
- Admin dashboard and analytics

### Quick Start
```bash
docker-compose up -d
go run cmd/api/main.go
```
